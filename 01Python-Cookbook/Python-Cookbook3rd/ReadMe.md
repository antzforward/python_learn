## 学习《Python-CookBook：Recipes for Mastering Python 3》 3rd Edition
目标：把这本书当作学习语法的初始版本，通过PyCharm工具，按照书的顺序，逐一完成测试代码，

一定要特别的显示出来，增加对语法的认知。
